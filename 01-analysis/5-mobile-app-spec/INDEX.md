# Индекс API-спецификаций и логических компонентов

Этот индекс служит навигационной картой для всех артефактов, связанных с API и бизнес-логикой клиентского приложения «Вертикаль».

> **Назначение:** Обеспечить централизованную точку входа для понимания архитектуры, API-контрактов и ключевых бизнес-процессов.
> 
> **Принцип:** Каждый файл описывает либо отдельный API-эндпоинт, либо атомарный бизнес-сценарий.

---
