# Фильтрация слотов

> **ID:** LOGIC-005
> **Тип:** Бизнес-логика / Алгоритм
> **Приоритет:** P1
> **Связанные требования:** FR-010, FR-014; UC-003; US-010

---

## 1. Назначение

Логика фильтрации списка слотов на экране расписания. Позволяет клиенту отфильтровывать слоты по формату тренировки и переключаться между днями недели.

**Сценарии:**
- Переключение между днями (горизонтальный скролл)
- Фильтрация по формату тренировки (все / болдеринг / трассы)
- Сохранение выбранного фильтра при переключении дней
- Комбинированная фильтрация (день + формат)

---

## 2. Исходные данные

### 2.1. Данные API

| Поле | Тип | Описание |
|------|-----|----------|
| `slots` | array | Полный список слотов от API |
| `slot.format` | enum | Формат тренировки |
| `slot.startTime` | datetime | Время начала тренировки |

### 2.2. Доступные фильтры

| Фильтр | Тип | Значения |
|--------|-----|----------|
| **Формат** | Enum | Все форматы, Болдеринг для новичков, Трассы с верёвкой для опытных |
| **День** | Date | Сегодня, Завтра, ПН, ВТ, ... (до 7 дней) |

| Значение API | Отображение в UI | Ключ фильтра |

| `BOULDERING_BEGINNER` | "Болдеринг для новичков" | `BOULDERING` |

## 5. DaySelector (навигация по дням)

### 5.1. Структура дней

| День | Отображение |
|------|-------------|
| Сегодня | "СЕГОДНЯ" |
| Завтра | "ЗАВТРА" |
| День 3 | "ПН 14" |
| День 4 | "ВТ 15" |
| День 5 | "СР 16" |
| День 6 | "ЧТ 17" |
| День 7 | "ПТ 18" |

### 5.2. Логика формирования

| Правило | Описание |
|---------|----------|
| **Сегодня** | Отображается как "СЕГОДНЯ" независимо от дня недели |
| **Завтра** | Отображается как "ЗАВТРА" |
| **Остальные дни** | Отображаются как "{День недели} {Число}" |
| **Активный день** | Подсвечивается оранжевым фоном |

### 5.3. Горизонтальный скролл

| Свойство | Описание |
|----------|----------|
| **Количество дней** | Всегда 7 дней (текущий + 6 вперёд) |
| **Начальная позиция** | Сегодня (первый элемент) |
| **Скролл** | Плавный, с возможностью пролистывания в обе стороны |
| **Автоцентрирование** | Активный день всегда в центре видимой области |

## 7. Матрица фильтрации

| День | Формат | Результат |
|------|--------|-----------|
| Сегодня | Все форматы | Все слоты на сегодня |
| Сегодня | Болдеринг | Только болдеринг на сегодня |
| Сегодня | Трассы | Только трассы на сегодня |

| Завтра | Болдеринг | Только болдеринг на завтра |
| ПН 14 | Все форматы | Все слоты на 14-е |

## 8. Поведение при переключении

### 8.1. Переключение дня

| Действие | Реакция |
|----------|---------|
| Нажатие на день | День становится активным, список обновляется |
| Горизонтальный скролл | Дни прокручиваются, активный день остаётся в центре |
| Pull-to-refresh | Обновление данных с API, сброс к активному дню |

### 8.2. Переключение формата

| Действие | Реакция |
|----------|---------|
| Нажатие на фильтр | Открытие выпадающего списка |
| Выбор формата | Закрытие списка, обновление списка слотов |
| Выбор "Все форматы" | Сброс фильтра, показ всех слотов |

---

## 9. Empty State

### 9.1. Условия отображения

| Условие | Состояние |
|---------|-----------|
| Нет слотов на выбранный день | "На этот день тренировок нет" |
| Нет слотов в выбранном формате | "Нет тренировок в этом формате" |
| Нет слотов на весь горизонт | "Пока нет доступных тренировок" |

### 9.2. Empty State для разных сценариев

| Сценарий | Заголовок | Подзаголовок | Кнопка |
|----------|-----------|--------------|--------|
| Нет слотов на день | "На этот день тренировок нет" | "Загляните на другой день" | — |
| Нет слотов по фильтру | "Нет тренировок в этом формате" | "Попробуйте выбрать другой формат" | "Сбросить фильтр" |
| Нет слотов вообще | "Пока нет доступных тренировок" | "Загляните позже — расписание обновляется" | "Обновить" |

---

## 10. Критерии приёмки

1. По умолчанию выбран сегодняшний день и фильтр "Все форматы"
2. При переключении дня список обновляется корректно
3. При выборе формата список фильтруется корректно
4. Фильтр по формату сохраняется при переключении дней
5. Dropdown показывает выбранный формат в заголовке
6. При отсутствии слотов показывается Empty State
7. При отсутствии слотов по фильтру показывается Empty State с кнопкой "Сбросить фильтр"
8. Горизонтальный скролл дней работает плавно
9. Активный день подсвечивается оранжевым фоном
10. Все названия форматов отображаются на русском языке
