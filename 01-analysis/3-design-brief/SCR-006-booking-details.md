# Экран деталей записи

> **ID:** SCR-006
> **Приоритет:** P0
> **Связанные требования:** FR-031, FR-032, FR-033, FR-034, FR-035, FR-040, FR-041, FR-042, FR-043; BR-5, BR-6; US-031, US-032, US-040, US-041, US-042; UC-005, UC-006, UC-008, UC-009, UC-010

---

## 1. Назначение экрана

Подробная информация о конкретной записи клиента. Позволяет управлять записью: отменить, изменить прокат (до порога `n` минут), а также видеть изменения со стороны скалодрома.

**Сценарии:**
- Клиент просматривает детали своей записи
- Клиент отменяет запись (с учётом порога `n` минут)
- Клиент изменяет выбор проката (до порога `n` минут)
- Клиент видит отмену скалодромом с причиной
- Клиент видит обновлённые данные при смене инструктора/времени

## 3. Компоненты экрана

### 3.1. Информационная карточка записи

| Элемент | Описание |
|---------|----------|
| **Время** | Дата и время тренировки (человекочитаемый формат) |
| **Формат** | Тип тренировки (с иконкой) |
| **Инструктор** | Имя инструктора |
| **Зона** | Название зоны |
| **Статус** | Цветной индикатор + текст статуса |
| **Цена** | Стоимость тренировки |

### 3.2. Секция проката

| Элемент | Описание |
|---------|----------|
| **Radio «Со своим снаряжением»** | Выбор без проката |
| **Radio «Взять в прокат»** | Выбор с прокатом |
| **Чекбоксы** | Скальники, страховочная система (с ценами) |
| **Итого** | Сумма стоимости проката |
| **Кнопка «Сохранить изменения»** | Появляется только при изменении выбора |
| **Предупреждение** | Появляется, если изменение проката недоступно (после порога `n` минут) |

### 3.3. Информация о пороге отмены

⏰ До начала тренировки осталось 2 часа 15 минут
Отмена доступна без последствий до 17:30

| Элемент | Описание |
|---------|----------|
| **Таймер** | До начала тренировки (в человекочитаемом формате) |
| **Информация о пороге** | Время, до которого отмена освобождает место |

**После порога:**

⏰ До начала тренировки осталось 30 минут
⚠️ При отмене место в группе не освободится


### 3.4. Кнопки действий

| Кнопка | Тип | Условие отображения |
|--------|-----|---------------------|
| **«Отменить запись»** | Secondary (красный текст/рамка) | Статус «Подтверждена» |
| **«Оценить инструктора»** | Primary | Статус «Завершена» и оценка не оставлена |
| **«Перейти к расписанию»** | Primary | Статус «Отменена скалодромом» |
| **«Сохранить изменения»** | Primary (маленькая) | Изменён выбор проката |

## 4. Состояния экрана

| Состояние | Описание | Визуальное поведение |
|-----------|----------|---------------------|
| **Загрузка** | Данные загружаются | Skeleton screen |
| **Подтверждена (до порога)** | Активная запись, отмена и прокат доступны | Все элементы активны, отмена освобождает место |
| **Подтверждена (после порога)** | Активная запись, отмена доступна, но место не освобождается | Предупреждение о пороге, изменение проката заблокировано |
| **Отменена клиентом** | Клиент отменил запись | Показывается дата отмены (в карточке) |
| **Отменена скалодромом** | Запись отменена организатором | Причина отмены, кнопка перехода к расписанию |
| **Завершена** | Тренировка состоялась | Кнопка оценки инструктора |
| **No-show / долг** | Неявка без оплаты | Сумма долга, информация о блокировке |
| **Ошибка** | Ошибка загрузки | Сообщение об ошибке + кнопка «Обновить» |

## 5. Взаимодействия

| Действие | Реакция |
|----------|---------|
| **Изменение выбора проката** | Появление кнопки «Сохранить изменения» |
| **Нажатие «Сохранить изменения»** | Отправка запроса в API; кнопка в состоянии загрузки |
| **Нажатие «Отменить запись»** | Открытие диалога подтверждения |
| **Подтверждение отмены** | Отправка запроса в API; возврат на экран «Мои записи» |
| **Нажатие «Оценить инструктора»** | Переход на экран оценки инструктора |
| **Нажатие «Перейти к расписанию»** | Переход на экран расписания |

## 6. Обработка ошибок

|----------|----------------------|----------|

## 7. Навигация

Мои записи → Детали записи (по карточке / кнопке «Подробнее»)
Детали записи → Мои записи (кнопка «Назад»)
Детали записи → Оценка инструктора (по кнопке «Оценить инструктора»)
Детали записи → Расписание (по кнопке «Перейти к расписанию»)

---

## 8. Критерии приёмки

1. Отображается полная информация о записи
2. При статусе «Подтверждена» доступна отмена записи
3. При статусе «Подтверждена» доступно изменение проката (до порога `n` минут)
4. После порога `n` минут изменение проката заблокировано с предупреждением
5. При отмене показывается диалог подтверждения с информацией о последствиях
6. При статусе «Отменена скалодромом» отображается причина и кнопка перехода к расписанию
7. При статусе «Завершена» доступна кнопка оценки инструктора
8. При статусе «No-show / долг» отображается сумма долга
9. Поддержка тёмной темы (опционально)
