# Экран расписания

> **ID:** SCR-002
> **Приоритет:** P0
> **Связанные требования:** FR-010, FR-011, FR-012, FR-013, FR-014, FR-015, FR-090; BR-3, BR-5; US-010, US-011, US-090; UC-003

---

## 1. Назначение экрана

Просмотр доступных слотов на 7 дней вперёд с навигацией по дням и фильтрацией.

**Сценарии:**
- Клиент видит расписание на неделю (дефолт: 7 дней от текущего момента)
- Клиент переключает дни (горизонтальный скролл)
- Клиент видит слоты, отменённые скалодромом
- Клиент переходит к записи на выбранную тренировку

## 3. Компоненты экрана

### 3.1. Навигация по дням (Day Selector)

┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐
│СЕГОДНЯ│ │ЗАВТРА │ │ПН 14 │ │ВТ 15 │ │СР 16 │ │ЧТ 17 │ │ПТ 18 │
└──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘
[актив] [обычн] [обычн] [обычн] [обычн] [обычн] [обычн]

- **Горизонтальный скролл:** Можно пролистывать пальцем
- **Активный день:** Оранжевый фон, белый текст, скругление 20 pt
- **Обычный день:** Прозрачный фон, тёмно-серый текст
- **Текст:** «Сегодня», «Завтра», «ПН 14», «ВТ 15»...
- **Размер:** 48 pt высота

### 3.2. Фильтр форматов (Dropdown)

┌──────────────────────────────────┐
│ Все форматы ▼│ ← Название фильтра + стрелка
└──────────────────────────────────┘
│
├── Все форматы
├── Болдеринг для новичков
└── Трассы с верёвкой для опытных


- **Раскрывающийся список:** При нажатии открывается список опций
- **Выбранная опция:** Отображается в поле с галочкой
- **Стиль:** Как поле ввода, но без ввода текста

### 3.3. Карточка слота

См. детальное описание в `SCR-003-slot-card.md`.

**Варианты состояний:**

| Состояние | Фон | Индикатор мест | Кнопка |
|-----------|-----|----------------|--------|
| **Доступен (много мест)** | Белый | 🟢 Осталось 5 мест из 8 | Активна, оранжевая |
| **Доступен (мало мест, ≤3)** | Белый | 🟡 Осталось 2 места из 8 | Активна, оранжевая |
| **Мест нет** | Белый | 🔴 Мест нет | Неактивна, серая |
| **Отменён скалодромом** | Серый (50%) | ⚪ ОТМЕНЕН СКАЛОДРОМОМ | Скрыта/неактивна |

### 3.4. Day Divider (разделитель дня)

────────────────── СЕГОДНЯ, 14 ИЮЛЯ ──────────────────


- **Текст:** Верхний регистр, 14 pt, серый цвет
- **Линии:** С обеих сторон, серая, тонкая
- **Отступы:** 16 pt сверху и снизу

## 4. Состояния экрана

| Состояние | Описание | Визуальное поведение |
|-----------|----------|---------------------|
| **Загрузка** | Данные загружаются | Skeleton screen для карточек и дней |
| **С данными** | Есть слоты на выбранный день | Полный список карточек |
| **Нет слотов на день** | На выбранный день нет слотов | Empty state внутри дня: «На этот день тренировок нет» |
| **Нет слотов вообще** | Нет слотов на весь горизонт | Полный empty state экрана |
| **Ошибка** | Ошибка загрузки | Сообщение об ошибке + кнопка «Обновить» |

---

## 5. Взаимодействия

| Действие | Реакция |
|----------|---------|
| **Нажатие на день** | Загрузка слотов на выбранный день; день становится активным |
| **Горизонтальный скролл дней** | Плавная прокрутка; активный день в центре |
| **Нажатие на фильтр** | Открытие выпадающего списка; выбор опции → обновление списка |
| **Нажатие на карточку слота** | Переход на экран деталей слота (SCR-004-booking) |
| **Нажатие на «Записаться»** | Переход на экран деталей слота (SCR-004-booking) |
| **Pull-to-refresh** | Обновление списка слотов |
| **Скролл списка** | Плавная прокрутка; загрузка новых данных при необходимости |

---

## 6. Обработка ошибок

| Сценарий | Сообщение пользователю | Действие |
|----------|----------------------|----------|
| Ошибка загрузки | «Не удалось загрузить расписание. Проверьте интернет и попробуйте снова» | Кнопка «Обновить» |
| Нет слотов на день | «На этот день тренировок нет» | Empty state внутри дня |
| Нет слотов на неделю | «Пока нет доступных тренировок» | Полный empty state экрана |
| Нет интернета | «Нет подключения к интернету. Показываем сохранённые данные» | Баннер сверху |

---

## 7. Навигация

Главный экран → Расписание (по кнопке «Всё расписание»)
Расписание → Детали слота (по карточке / кнопке «Записаться»)
Расписание ← Главный экран (кнопка «Назад»)

---

## 8. Критерии приёмки

1. Отображаются слоты на 7 дней вперёд (дефолтно)
2. Слоты сгруппированы по дням с разделителями
3. Каждая карточка содержит: время, формат, инструктора, зону, свободные места, цену
4. При отсутствии слотов — empty state
5. Слоты, отменённые скалодромом, отображаются как недоступные с указанием причины
6. Навигация по дням (горизонтальный скролл) работает
7. Фильтр по форматам работает
8. Pull-to-refresh обновляет данные
9. Индикация загрузки (skeleton screen) отображается
10. Поддержка тёмной темы (опционально)
