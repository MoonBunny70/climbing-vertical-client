## 1. Назначение

Блок-схема, описывающая логику работы фильтров на экране расписания. Позволяет клиенту фильтровать слоты по форматам тренировок.

## 3. Состояния фильтра

| Состояние | Отображение | Описание |
|-----------|-------------|----------|
| **Все форматы** | `Все форматы ▼` | Показываются все доступные слоты |
| **Болдеринг** | `Болдеринг ▼` | Показываются только слоты с форматом «Болдеринг для новичков» |
| **Трассы** | `Трассы с верёвкой ▼` | Показываются только слоты с форматом «Трассы с верёвкой для опытных» |

---

## 4. Взаимодействия

| Действие | Реакция |
|----------|---------|
| **Нажатие на фильтр** | Открытие выпадающего списка |
| **Выбор формата** | Закрытие списка, обновление списка слотов |
| **Выбор «Все форматы»** | Сброс фильтра, показ всех слотов |
| **Повторное нажатие** | Закрытие списка (без изменений) |

---

## 5. Критерии приёмки

1. Фильтр отображается на экране расписания
2. По умолчанию выбран «Все форматы»
3. При выборе формата список слотов обновляется
4. При выборе «Все форматы» фильтр сбрасывается
5. Выбранный фильтр сохраняется при переключении дней
