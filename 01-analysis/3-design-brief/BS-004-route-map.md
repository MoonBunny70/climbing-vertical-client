# Карта навигации

> **ID:** BS-004
> **Тип:** Блок-схема
> **Приоритет:** P0
> **Связанные требования:** Все экраны

---

## 1. Назначение

Карта навигации приложения, показывающая связи между всеми экранами.

## 3. Связи между экранами

### Главный экран
| Экран | Действие |
|-------|----------|
| Расписание | Кнопка «Всё расписание» |
| Мои записи | Кнопка «Мои записи» |
| Абонемент | Кнопка «Абонемент» |
| Профиль | Нажатие на аватар |
| Детали слота | Нажатие на карточку слота |
| Детали записи | Нажатие на карточку «Следующая тренировка» |

### Расписание
| Экран | Действие |
|-------|----------|
| Детали слота | Нажатие на карточку слота / кнопку «Записаться» |
| Главный экран | Кнопка «Назад» |

### Детали слота
| Экран | Действие |
|-------|----------|
| Главный экран | Успешная запись → кнопка «На главную» |
| Мои записи | Успешная запись → кнопка «Мои записи →» |
| Расписание | Кнопка «Назад» |

### Мои записи
| Экран | Действие |
|-------|----------|
| Детали записи | Нажатие на карточку / кнопку «Подробнее» |
| Оценка инструктора | Кнопка «Оценить инструктора» |
| Главный экран | Кнопка «Назад» |

### Детали записи
| Экран | Действие |
|-------|----------|
| Мои записи | Кнопка «Назад» |
| Оценка инструктора | Кнопка «Оценить инструктора» |
| Расписание | Кнопка «Перейти к расписанию» (при отмене скалодромом) |

### Профиль
| Экран | Действие |
|-------|----------|
| Абонемент | Кнопка «Управление →» |
| Мои записи | Кнопка «Вся история →» |
| Настройки уведомлений | Пункт меню |
| Экран входа | Кнопка «Выйти из аккаунта» |
| Главный экран | Кнопка «Назад» |

### Абонемент
| Экран | Действие |
|-------|----------|
| Расписание | Кнопка «Записаться на тренировку» |
| Профиль | Кнопка «Назад» |
| Главный экран | Кнопка «Назад» |

### Оценка инструктора
| Экран | Действие |
|-------|----------|
| Мои записи | Кнопка «Назад» |
| Детали записи | Кнопка «Назад» |

### Настройки уведомлений
| Экран | Действие |
|-------|----------|
| Профиль | Кнопка «Назад» |
