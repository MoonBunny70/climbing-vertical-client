# Экран профиля

> **ID:** SCR-007
> **Приоритет:** P1
> **Связанные требования:** FR-006, FR-007; US-005; UC-002

---

## 1. Назначение экрана

Просмотр и редактирование данных клиента. Отображение личной информации, статуса клиента, истории записей и быстрый доступ к настройкам и абонементу.

**Сценарии:**
- Клиент просматривает свои данные (имя, телефон, фото)
- Клиент видит статус клиента (для будущей программы лояльности)
- Клиент видит историю записей (прошлые и будущие)
- Клиент переходит к настройкам уведомлений
- Клиент переходит к управлению абонементом

## 3. Компоненты экрана

### 3.1. Карточка профиля (шапка)

| Элемент | Тип | Размер | Описание |
|---------|-----|--------|----------|
| **Аватар** | Изображение | 96×96 pt | Фото клиента (или иконка по умолчанию) |
| **Имя** | Текст | 22pt, Bold | Имя клиента |
| **Телефон** | Текст | 16pt, Regular | Контактный номер |
| **Email** | Текст | 16pt, Regular | Email клиента (если указан) |
| **Статус клиента** | Текст + иконка | 14pt, Regular | «🏅 Постоянный клиент» (без логики) |

### 3.2. Карточка абонемента

┌─────────────────────────────────────────────────────────────────┐
│ 📋 Абонемент │ ← Заголовок
│ Осталось занятий: 5 из 10 │ ← Информация
│ Действует до: 31.12.2026 │
│ [ Управление → ] │ ← Кнопка
└─────────────────────────────────────────────────────────────────┘

| Элемент | Описание |
|---------|----------|
| **Заголовок** | «📋 Абонемент» |
| **Остаток занятий** | «Осталось занятий: X из Y» |
| **Срок действия** | «Действует до: ДД.ММ.ГГГГ» |
| **Кнопка** | «Управление →» — переход к управлению абонементом |

│ ├─────────────────────────────────────────────────────────┤ │
│ │ ❌ 10.07 Болдеринг Анна С. Неявка │ │

|---------|----------|
| **Заголовок** | «📅 История записей (X)» — X = общее количество |
| **Строка записи** | Иконка статуса + дата + формат + инструктор + стоимость |
| **Иконка статуса** | ✅ (оплачено/посетил) / ❌ (неявка) |
| **Кнопка** | «Вся история →» — переход к экрану моих записей |

## 4. Состояния экрана

| Состояние | Описание | Визуальное поведение |
|-----------|----------|---------------------|
| **Загрузка** | Данные загружаются | Skeleton screen |
| **С данными** | Профиль загружен | Полный интерфейс |
| **Редактирование** | Пользователь редактирует данные | Поля активны, кнопка «Сохранить» активна |
| **Сохранение** | Данные сохраняются | Кнопка «Сохранить» в состоянии загрузки |
| **Ошибка** | Ошибка загрузки/сохранения | Сообщение об ошибке + кнопка «Обновить» |

## 5. Взаимодействия

| Действие | Реакция |
|----------|---------|
| **Нажатие на аватар** | (Если реализовано) Открытие галереи для выбора фото |
| **Нажатие на «Управление →»** | Переход на экран управления абонементом (SCR-008) |
| **Нажатие на «Вся история →»** | Переход на экран моих записей (SCR-005) |
| **Нажатие на «Настройки уведомлений»** | Переход на экран настроек уведомлений (SCR-010) |
| **Нажатие на «Выйти из аккаунта»** | Диалог подтверждения выхода |
| **Подтверждение выхода** | Очистка сессии, переход на экран входа |

---

## 6. Обработка ошибок

| Сценарий | Сообщение пользователю | Действие |
|----------|----------------------|----------|
| Ошибка загрузки профиля | «Не удалось загрузить данные профиля» | Кнопка «Обновить» |
| Ошибка сохранения | «Не удалось сохранить изменения. Попробуйте позже» | Snackbar |
| Нет интернета | «Нет подключения к интернету. Проверьте соединение» | Snackbar |

---

## 7. Навигация

Главный экран → Профиль (по аватару)
Профиль → Управление абонементом (по кнопке «Управление →»)
Профиль → Мои записи (по кнопке «Вся история →»)
Профиль → Настройки уведомлений (по пункту меню)
Профиль → Экран входа (после выхода из аккаунта)
Профиль ← Главный экран (кнопка «Назад»)


---

## 8. Критерии приёмки

1. Отображается аватар, имя, телефон, email (если указан)
2. Отображается статус клиента (поле без логики)
3. Отображается карточка абонемента с остатком занятий и сроком действия
4. Отображается история записей (последние 4-5 записей) с индикацией статуса
5. Доступна кнопка перехода к управлению абонементом
6. Доступна кнопка перехода к полной истории записей
7. Доступна кнопка перехода к настройкам уведомлений
8. Доступна кнопка выхода из аккаунта (с подтверждением)
9. Поддержка редактирования имени, телефона, email
10. Поддержка тёмной темы (опционально)
