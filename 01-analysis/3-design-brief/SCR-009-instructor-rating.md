# Экран оценки инструктора

> **ID:** SCR-009
> **Приоритет:** P1
> **Связанные требования:** FR-060, FR-061, FR-062, FR-063, FR-064, FR-085; BR-13; US-060, US-061, US-062, US-083; UC-014

---

## 1. Назначение экрана

Оценка инструктора клиентом после тренировки. Позволяет поставить оценку в звёздах и оставить текстовый комментарий.

**Сценарии:**
- Клиент ставит оценку инструктору в звёздах (обязательно)
- Клиент добавляет текстовый комментарий (опционально)
- Клиент может добавить комментарий позже (в любой момент)
- Клиент не может изменить звёзды после отправки (только через администратора)

## 3. Компоненты экрана

### 3.1. Звёзды (интерактивные)

⭐ ⭐ ⭐ ⭐ ⭐

| Элемент | Описание |
|---------|----------|
| **Звёзды** | 5 звёзд, интерактивные (можно нажимать) |
| **Подсветка** | При наведении/нажатии — оранжевый цвет |
| **Число** | «(5)» — отображение выбранного количества звёзд |
| **Размер** | 36×36 pt каждая |

**Взаимодействие:**
- Нажатие на звезду → выбор рейтинга (1-5)
- При наведении → предварительная подсветка
- Обязательное поле (нельзя отправить без выбора)

### 3.2. Текстовое поле для комментария

┌─────────────────────────────────────────────────────────────────┐
│ Ваш комментарий (опционально) │ ← Label
│ ┌───────────────────────────────────────────────────────────┐ │
│ │ Очень понравилась тренировка! Анна — │ │ ← TextArea
│ │ замечательный инструктор, всё понятно объясняет. │ │
│ └───────────────────────────────────────────────────────────┘ │
│ 0/500 │ ← Счётчик символов
└─────────────────────────────────────────────────────────────────┘

- **Тип:** Многострочное текстовое поле (TextArea)
- **Максимум:** 500 символов
- **Счётчик:** Отображение количества введённых символов
- **Плейсхолдер:** «Поделитесь впечатлениями о тренировке...»

### 3.3. Информация об анонимности

ℹ️ Оценка будет опубликована от вашего имени


- **Иконка:** ℹ️
- **Текст:** 14pt, Regular, Серый
- **Постоянно отображается** на экране

### 3.4. Кнопки действий

| Кнопка | Тип | Условие отображения |
|--------|-----|---------------------|
| **«Оценить»** | Primary | Оценка не оставлена, звёзды выбраны |
| **«Добавить комментарий»** | Secondary | Оценка оставлена, комментария нет |
| **«Сохранить комментарий»** | Primary | Добавление/изменение комментария |

## 4. Состояния экрана

| Состояние | Описание | Визуальное поведение |
|-----------|----------|---------------------|
| **Начальное** | Оценка не оставлена | Звёзды неактивны (серые), кнопка неактивна |
| **Выбор звёзд** | Пользователь выбирает звёзды | Звёзды подсвечиваются оранжевым, кнопка становится активной |
| **Загрузка** | Отправка оценки | Кнопка в состоянии загрузки |
| **Успех** | Оценка сохранена | Переход в режим просмотра |
| **Ошибка** | Ошибка сохранения | Snackbar с сообщением об ошибке |
| **Просмотр** | Оценка уже оставлена | Звёзды неактивны, отображение комментария |
| **Добавление комментария** | Добавление комментария к существующей оценке | Текстовое поле активно, кнопка «Сохранить комментарий» |

## 5. Валидация

| Поле | Правила валидации | Сообщение об ошибке |
|------|-------------------|---------------------|
| Звёзды | Выбрано от 1 до 5 | «Пожалуйста, оцените инструктора» |
| Комментарий | Максимум 500 символов | «Комментарий не должен превышать 500 символов» |

---

## 6. Обработка ошибок

| Сценарий | Сообщение пользователю | Действие |
|----------|----------------------|----------|
| Ошибка сохранения оценки | «Не удалось сохранить оценку. Попробуйте позже» | Snackbar |
| Ошибка сохранения комментария | «Не удалось сохранить комментарий. Попробуйте позже» | Snackbar |
| Нет интернета | «Нет подключения к интернету. Проверьте соединение» | Snackbar |

## 7. Навигация

Мои записи → Оценка инструктора (по кнопке «Оценить инструктора»)
Детали записи → Оценка инструктора (по кнопке «Оценить инструктора»)
Уведомление → Оценка инструктора (по нажатию на push-напоминание)
Оценка инструктора ← Мои записи (кнопка «Назад»)
Оценка инструктора ← Детали записи (кнопка «Назад»)


---

## 8. Критерии приёмки

1. Отображается имя инструктора, которого оценивает клиент
2. Клиент может выбрать оценку в звёздах (от 1 до 5)
3. Звёзды подсвечиваются оранжевым при выборе
4. Кнопка «Оценить» активна только после выбора звёзд
5. Клиент может добавить текстовый комментарий (опционально)
6. Комментарий не может превышать 500 символов
7. После успешной отправки показывается режим просмотра
8. Клиент не может изменить звёзды после отправки
9. Клиент может добавить комментарий позже в любое время
10. Оценка публикуется от имени аккаунта клиента
11. Поддержка тёмной темы (опционально)
